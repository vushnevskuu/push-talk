# Репозиторий VoiceInsert

Основной **публичный** репозиторий: **https://github.com/vushnevskuu/push-talk**  
(лендинг, веб, документация и связка с нативным приложением в одном месте.)

## Клонирование

```bash
git clone https://github.com/vushnevskuu/push-talk.git
cd push-talk
```

## Сборка macOS и ZIP для сайта

См. `web/README.md`: положить **`VoiceInsert-macos.zip`** в **`web/public/`** для раздачи с лендинга (`/VoiceInsert-macos.zip`).

По желанию дублируй релизы на вкладке **Releases** того же репозитория.
