VoiceInsert

Hold a shortcut, speak, release—text goes into the focused field or into your Obsidian vault as Markdown. No cloud required for dictation.

Download for Mac Source on GitHub
  • Works in any app with a text field—Safari, Notes, IDEs, and more.
  • Optional second shortcut saves structured notes under Voice Captures/ in Obsidian.
  • Microphone and speech stay on device; you control Accessibility and shortcuts in Settings.

Requirements

macOS 13+ · Apple Silicon or Intel (release ZIP matches the CI runner; build locally for your architecture if needed).

First launch: if Gatekeeper blocks the app, use Control-click → Open, or allow in System Settings → Privacy & Security. Ad hoc–signed builds behave like a local ./Scripts/build_app.sh run.

Permissions

Microphone, Speech Recognition, Accessibility (and Input Monitoring for some global shortcut modes). The app walks you through what is needed.
